⚙️ Configuração necessária

Para usar o sistema, você precisa:

  1. Abra a planilha Google Sheets e acesse Extensões → Apps Script
  2. Cole o conteúdo de apps-script/Code.gs e salve
  3. Clique em Implantar → Novo implante, tipo Aplicativo Web, acesso Qualquer pessoa
  4. Copie a URL gerada e cole em script.js na variável APPS_SCRIPT_URL
  5. Faça o push para o GitHub e recarregue esta página
